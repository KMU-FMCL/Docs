## 1 . 1 Service & API

### 1. **API** 의 중요성

- Front-end & Back-end 간 통신 수단
- Modern Web 에서 **API design** 의 중요성이 Website design 만큼 중요

### 2. **API** 의 역할

- Database schema[^2] 와 유사한 규약 역할
- **API** 정의와 수정, 중차대한 작업

##

[^API]: Application Programming Interface

[^2]: Database System 에서 구조, 구성 및 관계를 정의
